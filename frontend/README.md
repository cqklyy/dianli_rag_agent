# 电力交易智能问答系统 - 前端

## 项目简介

这是一个基于Vue 3 + Element Plus的电力交易智能问答系统前端项目，包含以下功能：

- **用户登录**：支持管理员和普通用户登录
- **智能问答**：基于SSE流式输出的问答界面
- **用户管理**：管理员可对用户进行增删查改操作

## 技术栈

- Vue 3
- Vue Router 4
- Pinia (状态管理)
- Element Plus (UI组件库)
- Axios (HTTP客户端)
- Vite (构建工具)

## 项目结构

```
frontend/
├── src/
│   ├── views/           # 页面组件
│   │   ├── Login.vue    # 登录页面
│   │   ├── Chat.vue     # 问答页面
│   │   └── UserManagement.vue # 用户管理页面
│   ├── router/          # 路由配置
│   ├── stores/          # 状态管理
│   ├── services/        # API服务
│   └── main.js          # 应用入口
├── package.json
├── vite.config.js
└── index.html
```

## 安装和运行

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

### 3. 构建生产版本

```bash
npm run build
```

## 后端API集成

前端通过代理配置连接到后端服务（默认端口5000）：

- 问答接口：`/api/chat/sse` (SSE流式接口)
- 用户管理接口：需要您在后端实现对应的用户管理API

## 注意事项

1. 当前用户管理功能使用模拟数据，需要您在后端实现对应的用户管理API
2. 登录验证目前为前端模拟，实际使用时需要连接后端认证接口
3. 确保后端服务在端口5000运行，否则需要调整vite.config.js中的代理配置
